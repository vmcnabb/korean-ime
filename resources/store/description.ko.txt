별도의 시스템 IME나 관리자 권한 없이 브라우저에서 한국어(한글)를 입력하세요.

한글 입력기를 사용하면 운영체제의 한국어 입력기를 설치할 수 없는 컴퓨터(직장, 학교 또는 공용 컴퓨터)에서도 한글을 입력할 수 있습니다. 표준 두벌식 자판을 사용하며, 물리적 키보드 설정(QWERTY, Dvorak, AZERTY, 러시아어 등)과 관계없이 작동합니다.


사용 방법

• 확장 프로그램 아이콘("a / 한" 버튼)을 클릭하거나 오른쪽 Alt 키를 눌러 로마자와 한글 입력을 전환하세요.
• 이제 평소처럼 한글을 입력하면 됩니다.


기능

• 표준 텍스트 상자, 텍스트 영역, 그리고 많은 온라인 서식 있는 텍스트 편집기(예: CKEditor)에서 한글을 입력할 수 있습니다.
• 화면 키보드 — 로마자와 한글 키를 모두 보여주는 떠 있는 키보드입니다. 확장 프로그램의 우클릭 메뉴에서 켜거나 끌 수 있습니다.
• 로마자 변환 — 한글 텍스트를 선택하고 우클릭한 다음 "로마자 변환"을 선택하면 로마자로 변환됩니다(국어의 로마자 표기법 기준이며, 일부 드문 예외는 처리되지 않습니다).
• 완성된 음절 다시 편집 — 커서를 음절 바로 뒤에 놓고 Shift+Backspace를 누르면 해당 글자의 조합을 다시 이어서 할 수 있습니다.


지원되지 않는 항목

• Google 문서 — 텍스트를 캔버스에 렌더링하고 확장 프로그램의 입력을 받지 않는 자체 편집기를 사용하므로 한글 입력이 작동하지 않습니다.
• 웹용 Microsoft Word — 같은 이유로 지원되지 않습니다.
• 브라우저 자체 페이지(chrome://…, about:…), 확장 프로그램/부가 기능 스토어, 그리고 다른 확장 프로그램의 페이지 — 브라우저가 모든 확장 프로그램의 실행을 차단합니다.
